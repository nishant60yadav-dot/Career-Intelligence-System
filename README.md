# Career Intelligence System (CIS)

## Overview

Career Intelligence System (CIS) is a Python-based automation project that searches career pages of selected organizations, collects relevant job opportunities, stores them in Excel, and uses ChatGPT to analyze and prioritize the best opportunities.

## Version 1 Goals

- Read target organizations from Excel
- Read search keywords from Excel
- Search career pages
- Collect matching jobs
- Remove duplicates
- Store results in Excel

ChatGPT is responsible for:

- Ranking opportunities
- CV optimization
- Cover letter recommendations
- Interview preparation
